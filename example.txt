  --->git config --global user.email

for tracking the particular project or folder git is must initialize
  --->git init
for watching the hidden folder in git
 --->ls -la
for tracking the particular file using git 
 --->git add <filename>    (git add index.js)
for tracking all file using git
--->git add .
for find which line is add or remove in file 
 --->git diff
for remove the file from git or stop the tracking file using git
 --->git rm <filePath>
taking an snapshorts of code along with time
 --->git commit -m <message>
 for watching the history of git commit (like time ,day,date,author,email ,message,commit Id)
   ---->git log
